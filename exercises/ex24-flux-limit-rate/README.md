# ex24-flux-limit-rate

## Enunciado
Implementa `limitRatePassThrough` para aplicar control de demanda.

## Que practicar
- `limitRate`

## Teoria breve
`limitRate` ajusta cuanta demanda solicita el consumidor al productor. Es una herramienta practica para controlar presion en pipelines y evitar picos de consumo.
