# ex09-mono-map-transform

## Enunciado
Implementa `decorate` para formatear un usuario.

## Reglas
- Aplicar `trim`.
- Pasar a mayusculas.
- Prefijar con `USER:`.

## Que practicar
- `map`

## Teoria breve
`map` es el operador base para transformaciones puras: mismo numero de emisiones, nuevo contenido. Es la forma reactiva de expresar conversiones de datos sin efectos secundarios.
