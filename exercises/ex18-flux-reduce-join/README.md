# ex18-flux-reduce-join

## Enunciado
Implementa `joinWithDash` para reducir un flujo a un string unido por `-`.

## Que practicar
- `reduce`

## Teoria breve
`reduce` combina elementos en un unico resultado final. Es equivalente a una agregacion funcional y se usa para producir resumenes o valores acumulados al terminar el flujo.
