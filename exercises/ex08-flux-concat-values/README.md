# ex08-flux-concat-values

## Enunciado
Implementa `appendDone` para agregar una marca final.

## Reglas
- Emitir todos los elementos de entrada.
- Al final emitir `DONE`.

## Que practicar
- `concatWithValues`

## Teoria breve
`concatWithValues` agrega elementos al final respetando orden y finalizacion de la fuente original. Sirve para senales finales, marcadores o valores de cierre de proceso.
