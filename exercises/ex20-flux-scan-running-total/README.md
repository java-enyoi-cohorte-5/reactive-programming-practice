# ex20-flux-scan-running-total

## Enunciado
Implementa `runningTotal` para emitir suma acumulada.

## Que practicar
- `scan`

## Teoria breve
`scan` emite acumulados parciales en cada paso, a diferencia de `reduce` que solo entrega el valor final. Es ideal para progresos, metricas en tiempo real y estado incremental.
