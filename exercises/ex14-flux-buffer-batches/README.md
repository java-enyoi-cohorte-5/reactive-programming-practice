# ex14-flux-buffer-batches

## Enunciado
Implementa `batchToCsv` para agrupar en lotes de 2 y unir con coma.

## Que practicar
- `buffer`
- transformacion de lotes

## Teoria breve
`buffer(n)` acumula emisiones en listas de tamano fijo, transformando un flujo elemento a elemento en un flujo por lotes. Es clave para batch processing y optimizacion de I/O.
