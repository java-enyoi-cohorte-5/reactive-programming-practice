# ex27-imperative-to-reactive

## Enunciado
Convierte una logica imperativa de parseo+filtro+transformacion en pipeline reactivo.

## Reglas
- Parsear enteros desde strings.
- Ignorar entradas invalidas.
- Quedarte solo con pares.
- Multiplicar por 2.

## Que practicar
- `flatMap`
- composicion declarativa completa

## Teoria breve
Este ejercicio resume el paradigma: expresar una cadena de reglas como pipeline declarativo, con manejo de errores local y transformaciones puras. La meta es reemplazar bucles y estado mutable por composicion de operadores.
