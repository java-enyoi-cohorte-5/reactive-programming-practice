# ex10-flux-take-limit

## Enunciado
Implementa `firstThree` para limitar emisiones.

## Reglas
- Emitir solo los primeros 3 elementos.
- Completar inmediatamente despues.

## Que practicar
- `take`

## Teoria breve
`take(n)` corta la secuencia despues de `n` emisiones y completa. Es util para paginacion, throttling logico y proteccion contra flujos muy largos o infinitos.
