# ex15-mono-filter-blank

## Enunciado
Implementa `nonBlank` para filtrar textos en blanco.

## Reglas
- Trim.
- Si queda vacio, completar sin emitir.

## Que practicar
- `filter`

## Teoria breve
`filter` mantiene solo elementos que cumplen una condicion. En `Mono`, si no cumple, el flujo queda vacio; esto modela ausencia de dato sin excepciones ni `null`.
