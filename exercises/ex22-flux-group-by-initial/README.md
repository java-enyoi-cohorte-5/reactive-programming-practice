# ex22-flux-group-by-initial

## Enunciado
Implementa `groupByInitial` para agrupar palabras por inicial.

## Reglas
- Ignorar strings vacios.
- Clave: primer caracter en minuscula.

## Que practicar
- `groupBy`
- `collectList`

## Teoria breve
`groupBy` divide una secuencia en subflujos por clave. Es util para particionar eventos por categoria (usuario, tipo, region) antes de aplicar agregaciones por grupo.
